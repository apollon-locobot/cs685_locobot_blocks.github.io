INTRODUCTION

This is a project done as a part of the CS685 Autonomous Robotics course at George Mason University. We've wanted to work on a real robot and chose to work on the LoCoBot PX100, as the Interbotix APIs make it easier to work with. We've chosen a problem that incorporates the concepts of Inverse Kinematics, Control, Probabilistic Robotics, Tracking and Localization, SLAM taught in the course. The project was developed using ROS Noetic

LOCOBOT PX100

LoCoBot PX100, a.k.a "Low-Cost Robot", from Trossen Robotics starts from about $3000 and has the following features:

  • Create 3 from iRobot (base)
  • Intel NUC NUC8i3BEH Mini PC
  • Intel® RealSense™ Depth Camera D435
  • PincherX 100 Robot Arm (4 degrees of freedom)
  • RPLIDAR A2M8 360° Laser Range Scanner
  • MoveIt Support
